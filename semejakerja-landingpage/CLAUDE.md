# semejakerja-landingpage (v1, legacy)

The original public landing page for the Semeja Kerja WFC community. **Static HTML only — no build, no npm, no dependencies.** Open `index.html` directly in a browser to preview.

> Superseded by `../semejakerja-landingpage-v2/` (Next.js). Keep this only for reference/quick edits; new landing-page work belongs in v2.

## Stack

- Single `index.html` + **Tailwind CSS via CDN** (config inlined in a `<script>` tag in the file) + a little vanilla JS for parallax/interactivity.
- Assets in `assets/`.

## Working here

- There is nothing to install or run — edit `index.html` and refresh the browser.
- Tailwind config (brand colors etc.) is defined inline in the page, not in a config file. Brand palette: `primary #3D155F`, `secondary #B5A566`, `accent #F0C733`, `text-light #F0E6D0`, `background #F9FAFB`.
- Content is Indonesian; keep copy consistent with the community voice.
